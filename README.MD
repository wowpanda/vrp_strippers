# Strippers script

## English

This script will allow you to call dancers and send them back for a certain amount. Everything is fully configurable: currently the script is configured for the Unicorn and for Night Club DLC After Hours.

### Installation

 1. Add * "vrp_strippers" * in the ** "resources" ** of your server
 2. In the "server.cfg" add: `start vrp_strippers`
 3. Add your different places in `client / config.lua` and` server / main.lua`
 4. Enjoy the script!

Do not hesitate to report if you encounter bugs! I still avaible right here on GitHub!
